#### Learn go with tests ![](red-green-blue-gophers-smaller.png)
#### Install on openSUSE ``` zypper in go ```
#### Hello World ```go //hello.go package main import "fmt" func main() { fmt.Println("Hello, world") } ``` ``` go run hello.go ```
#### How to test ```go //hello.go package main import "fmt" func Hello() string { return "Hello, world" } func main() { fmt.Println(Hello()) } ``` ```go //hello_test.go package main import "testing" func TestHello(t *testing.T) { got := Hello() want := "Hello, world" if got != want { t.Errorf("got '%s' want '%s'", got, want) } } ``` ``` go test ```
